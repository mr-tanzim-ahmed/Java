Weather App

#Overview
  This is a simple Java-based Weather App that fetches real-time weather data and air quality information using the OpenWeatherMap API. It features a graphical user interface (GUI) built with Swing and     displays weather details in a structured table format.

#Features
  -Fetches weather details based on the city name.
  -Displays temperature, humidity, pressure, and air quality index (AQI).
  -Shows current date and local time.
  -Provides AQI ratings from "Good" to "Very Poor".

#Technologies Used
  -Java (Swing for GUI)
  -OpenWeatherMap API
  -JSON parsing (Google Gson)

#How to Run
  -Obtain an API key from OpenWeatherMap.
  -Replace apiKey in the code with your API key.
  -Compile and run the Java program using NetBeans or any Java IDE.
  -Requirements
  -Java Development Kit (JDK)
  -Internet connection to fetch weather data
